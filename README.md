# Cyber-Nexus: The Last Node

Game *Action-Shooter* 2D bertema futuristik yang dibangun menggunakan bahasa pemrograman Python dan pustaka Pygame.

## Fitur Utama
- **Sistem Pertempuran**: Tembak musuh menggunakan kursor mouse.
- **Sistem Kesulitan**: Musuh akan bertambah banyak seiring bertambahnya jumlah *kill*.
- **Efek Visual**: Animasi ledakan partikel saat musuh hancur.
- **Sistem Nyawa**: Lindungi pesawat Anda dari serangan musuh!

## Cara Bermain
1. Pastikan Python sudah terinstal di komputer Anda.
2. Instal kebutuhan sistem:
   ```bash
   pip install -r requirements.txt
